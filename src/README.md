###样式命名规范

采用BEM规范命名，即`block__element_modifier`，举例：`.app-menu__item_active`

命名空间（前缀）：
- `l-` 布局模块(layout)，如：`l-container`
- `c-` 组件(component)， 如：`c-card`
- `h-` 辅助类(helpers)， 如：`h-clearfix`
- `t-` 主题(themes)，如：`t-dark`
- `s-` 特定作用域或上下文(scope...
- `is-`和`has` 状态
- `js-` js hooks
